Your To-Do List

    Info

    This is a simple To-Do List application. You can add, complete, and delete tasks. All tasks are saved in your browser's local storage, so they won't disappear when you reload the page.

    Mark tasks as completed using the checkbox, and delete tasks with the red ✖ button. Stay productive and keep track of what matters!

    About

    LifeHub To-Do List was created by me as a practice project to learn HTML, CSS, and JavaScript. It demonstrates DOM manipulation, localStorage usage, and dynamic UI updates.
